# 🚀 Personal Portfolio Website

A modern, responsive, and professional **developer portfolio website** built using **React**, **Vite**, and **Tailwind CSS**.  
This portfolio showcases my skills, projects, and experience as a **Full Stack MERN Developer**.

---

## 👨‍💻 About Me

I’m **Rameshwar Mane**, a final-year Information Technology student passionate about building scalable, production-ready web applications using modern technologies like **React, Node.js, Express, and MongoDB**.

This portfolio is designed to give recruiters a clear overview of my technical skills, real-world projects, and contact information.

---

## ✨ Features

- 🌙 Dark theme modern UI  
- 📄 Multi-page layout with React Router  
- ⚡ Fast performance using Vite  
- 🎨 Styled with Tailwind CSS  
- 🎥 Smooth animations using Framer Motion  
- 📱 Fully responsive (mobile + desktop)  
- 📎 Resume download support  
- 🚀 Deployed on Vercel  

---

## 🛠 Tech Stack

**Frontend**
